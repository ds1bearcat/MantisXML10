Enabling Browser Scripting

In Netscape:
  1. Select EDIT from the menu
  2. Select PREFERENCES... from the drop down
  3. In the Preferences dialog box, click the word ADVANCED (Do not click the + sign)
  4. Check the box labeled ENABLE JAVASCRIPT
  5. Select OK to clear both dialog boxes, close this window and select "After scripts are enabled, please load the home page."
In Internet Explorer:
Windows
  1. Select TOOLS from the menu
  2. Select INTERNET OPTIONS... from the drop down
  3. Select the SECURITY tab
  4. Select the CUSTOM LEVELS button
  5. Scroll down to find a selection called SCRIPTING
  6. Select the ENABLE radio button under ACTIVE SCRIPTING
  7. Select OK to clear both dialog boxes, close this window and select "After scripts are enabled, please load the home page."
Macintosh
  1. Select EDIT from the menu
  2. Select PREFERENCES... from the drop down
  3. Scroll down to find a selection called WEB CONTENT
  4. Check the box labeled ENABLE SCRIPTING
  5. Select OK to clear the dialog boxes, close this window and select "After scripts are enabled, please load the home page."
