Platforms

Web/UI Technologies

MANTIS employs a variety of open web technologies to create rich portable interfaces:
  • HTML/HTML5

Databases/Interfacing

As well as supporting many major databases (ODBC, SQL, Supra, DB2, Oracle) MANTIS includes an Internal File key-based storage system. MANTIS also makes it easy to interface with other software products through XML design utilities.

Language

The MANTIS programming language is an easy, compact, and fast executing interpreted and pre-compiled scripting language.
  • Human Readable
  • Non-verbose
  • Easy Object Manipulation
  • Platform and Hardware Independent
  • Automatic Memory Management
